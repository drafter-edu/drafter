# Components Cheatsheet

This is a cheatsheet for the Components in Drafter. For a more detailed listing, check out: [components_detailed]

  --------------------------------------------------------------------------------
  Component Function                        HTML Tag
  ----------------------------------------- --------------------------------------
  Image(url)                                [img]{.title-ref}

  Image(url, width, height)                 [img]{.title-ref}

  Link(text, url)                           [a]{.title-ref}

  TextBox(name)                             [input]{.title-ref}

  TextBox(name, default_value)              [input]{.title-ref}

  TextArea(name)                            [textarea]{.title-ref}

  TextArea(name, default_value)             [textarea]{.title-ref}

  SelectBox(name, options)                  [select]{.title-ref}

  SelectBox(name, options, default_value)   [select]{.title-ref}

  CheckBox(name)                            [input]{.title-ref}

  CheckBox(name, default_value)             [input]{.title-ref}

  FileUpload(name)                          [input]{.title-ref}

  LineBreak()                               [br]{.title-ref}

  HorizontalRule()                          [hr]{.title-ref}

  Button(text, url)                         [button]{.title-ref}

  Button(text, url, arguments)              [button]{.title-ref}

  NumberedList(items)                       [ol]{.title-ref}

  BulletedList(items)                       [ul]{.title-ref}

  Header(body)                              [h1]{.title-ref}

  Header(body, level)                       [h1]{.title-ref} to [h6]{.title-ref}

  Table(data)                               [table]{.title-ref}

  Span(\...components)                      [span]{.title-ref}

  Div(\...components)                       [div]{.title-ref}

  Row(\...components)                       [div]{.title-ref}

  PreformattedText(text)                    [pre]{.title-ref}

  Argument(name, value)                     [input]{.title-ref}

  MatPlotLibPlot()                          [img]{.title-ref}

  Download(text, filename, contents)        [a]{.title-ref}
  --------------------------------------------------------------------------------
